### Perfil Profesional  
Ingeniero Civil Industrial especializado en el diseño, la validación técnica y el escalamiento de soluciones de datos para entornos industriales e infraestructura crítica (minería, energía y servicios regulados). Ocho años de experiencia diseñando y liderando implementaciones ETL, integración de datos y desarrollo de modelos para la optimización de procesos y mitigación de riesgos tecnológicos. Enfoque tecno-agnóstico y sistémico, orientado a la estandarización de procesos, la gobernanza de datos y el modelado de sistemas complejos.
